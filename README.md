# decorative-floor-registers

Premium e-commerce website for Decorative Floor Register — decorativefloorregister.com

## Tech Stack

- **Frontend:** Next.js 15 (App Router) + Tailwind CSS v4 + Framer Motion
- **Database:** Supabase (Postgres + Auth + Storage)
- **Payments:** Stripe Checkout
- **Email:** Resend
- **Deployment:** Vercel
